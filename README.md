# Shopping List App for a Technical Interview

## Summary

shopping-list uses React, TypeScript, and MUI.

View a live demo [here](https://shopping-cart-app.camcobb.com/)!

Features: 
- Add items
- Remove items
- Edit items
- Dark mode!
- Mobile friendly!

## Getting Started

1. Clone the repo
2. `npm install`
3. `npm start`
4. Enjoy!
